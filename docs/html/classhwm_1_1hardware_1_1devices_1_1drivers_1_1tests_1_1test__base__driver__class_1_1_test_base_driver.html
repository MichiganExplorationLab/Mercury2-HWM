def hwm.hardware.devices.drivers.tests.test_base_driver_class.TestBaseDriver.test_writing_device_output (   self)

Tests that the Driver class can pass its output to its registered pipelines.

The default implementation of the Driver.write_output() method only writes to active pipelines that specify this device as its output device.

def hwm.hardware.devices.drivers.tests.test_base_driver_class.TestBaseDriver.test_writing_device_telemetry (   self)

Tests that the Driver class can pass device telemetry and extra data streams to its registered pipelines via the Driver.write_telemetry() method.

This method should always be used to write extra device data and telemetry back to its pipelines.
