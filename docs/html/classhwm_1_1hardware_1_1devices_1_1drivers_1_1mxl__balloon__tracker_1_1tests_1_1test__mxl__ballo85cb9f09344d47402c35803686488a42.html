def hwm.hardware.devices.drivers.mxl_balloon_tracker.tests.test_mxl_balloon_tracker.TestAPRSTrackingService.test_aprs_query (   self)

Tests that the APRS query method can successfully query the APRS API for the balloon's position and parse the response.

Note
This test uses a mocked network handler instead of actually querying the APRS.fi API. As a result, if the API changes this test will still pass even though it will fail in production. This was done to limit test dependencies.
