Detailed Description

A driver for the custom MXL antenna controller.

This HardwareDriver class provides an interface to the MXL antenna controller.

Updates the antenna controller's state. 

 This method queries the antenna controller for its current azimuth and elevation and saves it to the driver state.
 It should typically be called automatically using something like LoopingCall.
Returns
Returns a dictionary containing the new device state. If an error occurs updating the state None will be returned instead.

def hwm.hardware.devices.drivers.mxl_antenna_controller.mxl_antenna_controller.MXL_Antenna_Controller.prepare_for_session (   self,
  session_pipeline 
)

Prepares the antenna controller for use by a new session.

Note
This method loads the active 'tracker' service from the session pipeline. If no such service is available, the antenna controller will still respond to manual adjustments but it will not be able to automatically track a target.
Parameters
session_pipelineThe Pipeline associated with the new session.
Returns
Returns a deferred for the state update LoopingCall if a 'tracker' service can be loaded and False otherwise.

Instructs the antenna controller to point at the specified target. 

 This callback commands the antenna to point at the specified target's azimuth/elevation. It is called every time the
 loaded "tracker" service has new position information.
Exceptions
Mayraise InvalidTargetPosition if the specified target position does not contain an azimuth or elevation.
Parameters

Returns
