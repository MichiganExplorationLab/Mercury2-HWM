Detailed Description

This test suite verifies the functionality of the ICOM 910 driver.

Note
The ICOM 910 driver depends on Hamlib and its Python bindings. To minimize testing dependencies, Hamlib APIs will be mocked when possible during the tests in this module (although the Hamlib module is still required).
