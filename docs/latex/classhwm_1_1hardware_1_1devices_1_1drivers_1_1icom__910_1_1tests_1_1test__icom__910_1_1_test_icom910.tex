\subsection{Detailed Description}
This test suite verifies the functionality of the I\-C\-O\-M 910 driver. 

\begin{DoxyNote}{Note}
The I\-C\-O\-M 910 driver depends on Hamlib and its Python bindings. To minimize testing dependencies, Hamlib A\-P\-Is will be mocked when possible during the tests in this module (although the Hamlib module is still required). 
\end{DoxyNote}
